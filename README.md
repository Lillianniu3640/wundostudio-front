# Illustration portfolio website
## 前台
1.Gsap 動畫<br>
2.RWD設計<br>
3.瀏覽作品<br>
4.網站風格規劃<br>

## 管理者後台
1.管理者註冊/登入<br>
2.作品增/改/刪/查<br>
3.資料存取：MongoDB + Mongoose 套件<br>

1.使用 Vue cli 開發<br>
2.Axios 串接資料庫資料<br>
3.RWD 響應式網站<br>
4.Google Analytics 分析<br>
5.Github/Heroku 網站部署<br>
